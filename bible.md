## 二、世界观基础【锁定】

- **时代背景**：架空古代，沿海渔村 + 地下囚所 + 深海神域三层空间。
- **核心设定**：深海鲛族（非人神明）曾庇佑沿海人族；人族以血脉气运为代价向鲛族借力，久而久之成"供养关系"。
- **事件起点**：鲛王（溟昭）违逆族规救了渔民（含沈家先祖），留陆后被沈家以邪术反噬封印，囚于地底十余年。
- **叙事视角**：以女主阿九（今生）为操控视角，前世明微记忆碎片穿插。

## 三、核心人物设定

### 3.1 男主一：溟昭·鲛王（主线核心）【锁定】
> **正式定名**：溟昭（溟=北冥大海，昭=幽海之光）。第一集中全程以"他/怪物/祸祟"指代，名字本身是"怪物→神"反转的揭示筹码，剧本层面留至后期揭名。

| 属性 | 内容 |
|------|------|
| 身份 | 深海鲛族王，顶级掠食者，曾庇佑沿海百年 |
| 外形关键词 | 濒死也散发致命压迫感；鳞裂如刀痕；骨相极美 |
| 人设矛盾 | 狂暴掠食者 ↔ 为她（唯一）崩塌的隐忍 |
| 囚禁状态 | 地牢十余年，神力枯竭但骄傲未折 |
| 产出机制 | **血凝赤珠**（见第五章） |
| 前世记忆 | 对前世明微有执念，认出阿九靠"善意"而非脸 |
| 核心悲剧 | 他的牢，全是他的善良砌的（见三重囚笼） |

### 3.2 女主：阿九（今生）/ 明微（前世）【锁定】

| 属性 | 内容 |
|------|------|
| **今生身份** | 沈家最底层粗使丫头"阿九"，记忆被抹去，来历不明——**讽刺真相：她正是明微今世转世，前世沈家嫡女，今生沦为沈家奴** |
| **前世身份** | **沈明微，沈家嫡女**。偶然发现家族密辛，得知家族百年繁华皆源于对他的囚禁与榨取，从小的衣食无忧都建立在他的血与痛上，心生愧疚与恻隐，走上秘密救助之路 |
| **前世行动** | 私下带食物、为他疗伤、月复一月积攒机会，终于找到时机助他出逃；逃亡途中被家族追上，他被重新押回，**明微以凄惨下场收场，堕入轮回** |
| **人设核心** | **不是被动棋子**；她的善意是主动选择，不是命运安排——她知道代价，仍然去做 |
| **与男主关系** | 她无意识重复了前世的善意动作（递水、护水）——他认出的不是脸，**是那份在这个炼狱里绝不可能出现的、不带目的的温柔** |
| **独立弧线** | 今生：从懵懂的恻隐 → 逐渐拼回前世记忆碎片 → 主动承担解封代价（斩断沈家气运链）；她的成长弧与他的解封弧并行，非工具人 |
| **身份命运讽刺** | 前世贵为沈家嫡女，今生沦为沈家奴；前世因他而死，今生被安排在他身旁却不记得为何——**命运把她送回来，是来偿还，还是再死一次？** |

## 四、三重囚笼（为何神明无法自救）【锁定】

> 核心：**他不是弱，是善。他的牢全是他的善良砌的。**

### 第①重：离水枯竭（天性锁）
鲛人离开**活水/海水**便日渐枯竭。地牢虽阴寒，却是**死水**，无法恢复神力。陆地对他是永恒的干涸与超重——他每一天都在"溺死"，攒不起挣脱的力气。

### 第②重：邪术反噬（慈悲锁）——最毒
封印燃料正是**沈家世代承他庇佑所积的"气运/血脉"**。他若强行挣脱，邪术反噬——代价由他曾守护的整片土地承担。**他不是挣不开，是不忍挣开。** 神性中的"护佑本能"成了最温柔的牢笼。

> 后续伏笔：解封关键可能是阿九（沈家血脉）以某种方式"斩断气运链"。

### 第③重：双世遗弃（孤立锁）
深海王族有禁令：不得近陆、不得渡凡人。他救渔民、留陆地，是**违逆族规的"叛者"**。
- 族人视他为耻，不来援救；
- 邪术**斩断了他与海的血脉感应**，深海族以为他早已死了。
- 结果：**人类背叛了他，同类除名了他。** 茫茫天地，唯一把他当"存在"对待的，是一个被当牲口使唤的丫头。

> 后续伏笔：沧溟（深海王子）发现他没死，找上门来，触发族人线。

## 五、核心机制

### 5.1 血凝赤珠 vs 泪珠【锁定】

| 机制 | 内容 | 设计逻辑 |
|------|------|---------|
| **血凝赤珠（日常）** | 鲛王之血遇空气凝结成赤珠/血珀，价值连城。**越是干渴濒死，逼出的血珠品相越红越好**。 | 贴合掠食者气质；完美解释了为什么狱卒**绝不给他水喝**的虐待动机。 |
| **泪珠（唯一高光）** | 鲛人泪为绝品，十几年他从未落过一滴 | 骄傲不折；唯一一次落泪只为她（前世明微之死/今生阿九），是十几年神性的唯一崩塌 |

**记忆口诀**：血是他们抢来的矿，泪是他为你碎的防。

### 5.2 递水与滴血的真正意义【锁定】


**✅ 正确设定：**
- 囚禁者**养着他的"命"**（需要榨血），但**断绝他的水**（为了保证血珠品相）。
- 十余年的酷刑与孤独，正把他从"神"磨成嗜血的"**兽**"。
- 阿九递水的动作，是**十几年里唯一一份不带目的的善意**，但在今生，水被狱卒踢翻。
- **真正的觉醒契机（已更新）**：阿九偷喂水被狱卒阻断后，鬼使神差地哼出了祭海渔歌的残调（"坎坎击鼓，望极浦"）。他如狂兽暴起、扑到链极，却在听清调子的瞬间僵住——那是只有前世明微才知道的古老渔歌。他用破碎的嗓音接上了下句（"泣珠报恩，明年期"），随即退回黑暗。**声音认出了她，不是血。** 这一设计彻底移除"闻血暴起"套路，以声音/记忆为宿命锚点，宿命感更深、更私密。
- 物理暗线（不破坏三重锁）：阿九偶尔从**靠海的老井/旧海神祠**打水，水里有若有若无的"海"的气息——是慰藉，不是补给，亦埋她来历的伏笔。

## 六、前世今生闭环【锁定】

### 6.1 前世·明微完整叙事【锁定】

> 灵感来源：《黑神话：悟空》第三回动画《屁》——以自身为代价、渡人于囹圄，而后堕入轮回。

**时间线：**

```
沈家先祖被救  →  奉神明为庇佑  →  小渔村渐成富豪大族
      ↓
 沈家起贪念  →  寻得邪术  →  压制神力·囚禁·强制榨取
      ↓
明微（沈家嫡女）偶然发现家族密辛
 "我从小的衣食无忧，原来全是他的血换来的"
      ↓
 动了恻隐之心  →  秘密救助（送食·疗伤·月复一月）
      ↓
 终于找到机会，助他出逃·夜奔向海
      ↓
 逃亡途中·家族追至·他被押回
      ↓
【明微下场凄惨·堕入轮回】
      ↓
 他目睹明微的结局（画外，不直展示）
 十几年囚禁中，唯一一次落泪  →  白色泪珠·凝而不散
 那一滴泪，是他十几年囚禁中仅有的人性残存
```

**明微死亡的叙事价值（参照黑神话第三回美学）：**
- 她的死**不直接展示**——只看他的反应，只见他对着空处伸出手，然后手落下
- 这样的处理比直接展示更残忍，留白让玩家自己填满最坏的想象
- 这一刻也是他**被永久封印意志**的瞬间：失去她之后，他停止了反抗

### 6.2 今生·阿九的轮回【锁定】

```
前世明微死·堕入轮回  →  记忆被抹去·身份归零
      ↓
今生以"阿九"身份出现在沈家  →  最底层的粗使丫头
（命运讽刺：前世沈家嫡女，今生沈家奴）
      ↓
无意识重复前世善意（递水·护水）
      ↓
他认出的不是脸——是那份在炼狱里绝不该出现的温柔
      ↓
"你……又来了"  →  宿命闭环成立
```

**关键问题（驱动剧情）：**
命运把她送回来，是来还债，还是再死一次？

### 6.3 主线解封路径【锁定】

- **解封第①重（离水）**：阿九的水 → 情感慰藉 → 他的神智逐渐归位（起点）
- **解封第②重（邪术）**：阿九以沈家血脉主动斩断气运链 → 代价由她承担（高潮）
- **解封第③重（遗弃）**：沧溟出现 → 族人线触发 → 他是否选择归海（结局分支）
- **True End方向**【待议】：两世善意合一，他终于离水归海；她随行·或各安天命·或再入轮回

## 八、关键剧情设计

### 8.1 冷开场（第一集前12秒）【锁定】

**视觉钩子**：镜头特写→一面刻满"沈"字的石壁，墨迹深浅不一，有的已被凿掉重刻。一只手缓缓抚过其中一个字，停住。**拉镜**——镜头拉开，是地牢，是他。

**信息差设计**：观众不知道这些字是谁写的、为什么写、为什么那么多——**带着问题往下看**，信息鸿沟在后续剧情逐层填平。

### 8.2 渔歌关键选择（第一集核心交互）【锁定·已重设计】

**场景**：阿九被派去打扫牢房血污（带水桶），打扫时偷偷喂了他半瓢水，被狱卒撞破踢翻水桶后骂走。狱卒离开后，牢中只剩两人。

**触发时机**：阿九被派去打扫牢房血污，偷偷喂了他半瓢水，被狱卒踢翻水桶后当面揪发威胁。狱卒扔下"今晚少主来验珠、唯你是问"的炸弹后大步离开，**门虚掩不落锁**。牢中只剩两人。

| 选项 | 内容 | 游戏效果 |
|------|------|---------|
| A【保命】 | 快走，离这里越远越好 | 她走后铁链极轻绷紧一下（信息差仅给观众）；共鸣值不变 |
| B【不忍】 | 把桶里最后半瓢水再悄悄推过去 | 他死寂的眼第一次"对上焦"看她；**共鸣值+1**；人性微光线 |
| C【鬼使神差】 | 那支古调涌上喉咙，无意识哼出 | 他暴起→听清曲调僵住→接上下句→"你又来了"→阿九惊恐逃出；**共鸣值+2**，梦境碎片①解锁 |

**逻辑自洽**：门虚掩→三条线均可自然离开，根治旧版"落锁后怎么出去"的硬伤。

**设计核心**：三选项是"人性光谱"而非非此即彼，玩家在"保命/善良/宿命"之间撕裂；C线"夺门而出"补足了心理动机（不是它说话=它认得她，寒意+剧痛=宿命碰触）。

### 8.3 方言/声音母题【锁定】

方言承担三个叙事功能（绝不做全程对白，永远配字幕）：

| 功能 | 内容 |
|------|------|
| **阶级标记** | 沈家主子说官话，仆役说地方口音→身份一耳听出 |
| **身份暗线** | 鲛王偶尔说出只有前世明微听过的古老方言词汇→她莫名熟悉→前世记忆钩子 |
| **催泪声音母题** | 一首古老渔歌童谣，贯穿前世今生——今生阿九无意哼起，他骤然失神 |

## 九、游戏性系统设计【v1.0】

> 三个系统相互咬合，全部服务于"玩家自己挖出怪物=神"的核心叙事体验。

### 9.1 梦境碎片收集系统

| 属性 | 内容 |
|------|------|
| **触发方式** | 高共鸣值选择 / 场景调查特定物件 / 部分线索对话 |
| **碎片形态** | 无声黑白残像+一句残诗，单块残缺、拼合才见真相 |
| **每集数量** | 1-2块；部分碎片"可错过"，逼出二周目 |
| **拼合奖励** | 拼合进度直接揭示"怪物→神"反转真相，是玩家主动挖出来的，不是旁白告知 |
| **第①块** | 第一集·选项B后解锁："一双手/白珠"，余皆黑——只是开始 |

### 9.2 共鸣值系统（取代普通好感度）

| 属性 | 内容 |
|------|------|
| **命名逻辑** | 阿九与溟昭是前世同魂，情感度叫"共鸣"而非"好感"，绑定宿命主题 |
| **提升行为** | 哼渔歌、偷喂水、靠近、不逃跑、调查神明相关线索 |
| **降低行为** | 恐惧、举报、逃跑、配合狱卒虐待 |
| **解锁效果** | 共鸣↑ → 溟昭回应由"死寂"→"接曲"→"开口"→"护她"；解锁更多梦境碎片 |

### 9.3 道德资源：血珠 / 泪珠（生存vs人性）

| 资源 | 获取方式 | 消耗 | 道德含义 |
|------|---------|------|---------|
| **血珠（日常）** | 配合榨血流程，虐待/不阻拦 → 产量高 | 换取沈家信任/阿九安全 | 用他的痛换自己的命 |
| **泪珠（唯一）** | 高共鸣 True End 专属 | 解封第②重锁（邪术）的代价 | 两世善意兑现的承诺 |
| **核心张力** | 血珠多=安全但共鸣↓；善待=共鸣↑但有被抓风险。玩家每集都被撕裂 ||

### 9.4 场景调查（信息主动权给玩家）

- 场景内设**可点击物件**：沈字墙、血珠盒、鳃纹刺青、玉佩、海神祠香灰……
- 点击触发一句旁白或一段线索，玩家主动拼出世界观，不靠编剧旁白直给
- 直接解决"信息过于直给"的问题——**怪物=神的揭示，是玩家点出来的**

### 9.5 三系统咬合图

## 十二、人物小传（性格向·美术参考）

> 说明：本章重在**性格内核与行为逻辑**，外形仅给关键词，具体造型交由美术发挥。

### 12.1 溟昭（男主一·怪物→神）

- **性格内核**：骄傲、隐忍、护短到自毁。神性未泯，却被十余年酷刑磨出兽性——理智与本能在他体内日夜交战。
- **行为逻辑**：对所有人是死寂的敌意；唯独对"不带目的的善意"会失控。他认出阿九不是因为脸，是因为那份在炼狱里绝不该出现的温柔，他记得。
- **说话方式**：极少开口，一开口是破碎气声或只有前世才懂的古老残句。
- **反差点**：顶级掠食者，却会因一个丫头被揪头发而铁链绷紧——在意到极点，又克制到极点。
- **外形关键词**：苍白、骨相极美、鳞裂如刀痕、胸口鳃纹刺青（冻青＋靛蓝）。

### 12.2 阿九 / 明微（女主·今生·前世）

- **今生阿九**：底层粗使丫头，谨小慎微、习惯性低头；但骨子里有压不住的恻隐——善良是麻木生活里唯一没被磨掉的东西。
- **前世明微**：沈家嫡女，清醒、有担当，知道代价仍选择救他。**不是恋爱脑，是"看见了不公就装不了看不见"的人。**
- **核心矛盾**：今生她不记得前世，却总被莫名的反应（心悸、流泪、无意识哼歌）出卖——**身体比记忆诚实。**
- **成长弧**：从"怕事的丫头"→"主动追问真相、主动承担代价的人"。
- **外形关键词**：清瘦、眉眼带韧劲、手掌粗糙（劳役痕迹）。

### 12.3 沈砚（男主二·白月光→白切黑伏笔）

- **表层**：沈家少主，温润如玉、彬彬有礼，对下人也客气——第一集的"白月光"。
- **里层（伏笔）**：他是这套榨取体系最大的既得利益者。**他的温柔越完美，反转越冷。**
- **行为细节**：递帕子时那一眼扫向地牢——他什么都知道，只是选择体面地享用。
- **张力**：不是脸谱反派。他的"好"是真的，"恶"也是真的，二者并存才可怕。
- **外形关键词**：月白锦衣、干净、提琉璃灯、带冷香。

### 12.4 狱卒（配角·体系的爪牙）

- **功能**：把"神被当牲口"具象化的人，制造即时冲突与压迫。
- **性格**：欺软怕硬、贪婪、麻木；对怪物的恐惧用更凶的虐待来掩盖。
- **作用**：反衬阿九善意之珍贵；也是"血珠＝财气"规则的执行者。

### 12.5 买珠贵客（符号角色·冷开场的手）

- **功能**：冷开场那只手，3秒内把"血珠＝钱"的残酷链条砸到观众脸上。
- **形象**：戴满玉扳指、贪婪、对珠子的来源毫不在意——人对神之痛的彻底无视。
- **作用**：无台词、纯符号，开场即立"这个世界的冷酷规则"。

## 十三、第一集场景设定（美术参考）

### 13.1 验珠堂 / 卖珠场（冷开场）

- **氛围**：奢靡、暖光、人对财货的贪婪。与地牢的冷构成第一组对照。
- **关键道具**：赤色血珠、戴玉扳指的手、灯。
- **叙事功能**：用最快的方式立"血比命值钱"的世界规则。

### 13.2 极寒囚牢（主舞台·地牢）

- **氛围**：冷白光、阴湿死水、铁锈与血腥；压迫、绝望。
- **配色**：冻青 ＋ 靛蓝 ＋ 血红。
- **关键道具**：刻满"沈"字的石壁、铁链、水桶、血珠盒、鳃纹刺青。
- **叙事功能**：怪物登场、渔歌触发、三选项交互，全部发生于此。

### 13.3 月下庭院（沈砚登场）

- **氛围**：清冷、皎洁、月白；与地牢"地下污暗"形成上下垂直对照。
- **关键道具**：白玉石阶、琉璃灯、冷香帕子。
- **叙事功能**：沈砚登场，第一集即确立两男主的视觉/气质反差。

### 13.4 仆役下房（集尾钩子）

- **氛围**：逼仄、漏风、清贫——阿九的真实处境。
- **关键道具**：木板床、头顶漏风的缝隙。
- **叙事功能**：梦境碎片触发地、集尾"她从未唱过这支歌"钩子的发生地。
